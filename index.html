Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines style of CSS.
  • The base element signifies the base URL.
  • The meta element defines the metadata i.e. character set, description, keywords, author, and viewport.

CSS

  • CSS add style to webpage.
  • CSS should be added second after the framework of HTML.

Git

  • Git is version tracking software.
  • Github is one of the most common version tracking software.
  • Version tracking allows developers to collaborate while managing changes to a webpage and revert changes if needed.

JavaScript

  • Javascript is added to original HTML.
  • Javascript should be added third after CSS or styling.
  • Javascript makes the webpage interactive for the user.
